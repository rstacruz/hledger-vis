# hledger-vis

> Web UI for hledger

## Usage

```sh
npm install -g @rstacruz/hledger-vis
hledger-vis
```

## Dev setup

* Setup:

  ```sh
  git clone https://github.com/rstacruz/hledger-vis.git
  cd hledger-vis
  npm install
  ```

* Start in development mode (auto-reloads):

  ```sh
  npm start
  ```
